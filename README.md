# 🎨 Color Palette Generator
Generate beautiful color palettes from any base color.
## Features
- 4 color schemes: Analogous, Complementary, Triadic, Monochromatic
- Custom base color picker
- Random color generator
- Click any swatch to copy HEX code
- 5-color palette with hover expand effect
- Individual color code cards
## Built With
- HTML5
- CSS3
- Vanilla JavaScript
## How to Run
Clone the repo and open `index.html` in your browser.
